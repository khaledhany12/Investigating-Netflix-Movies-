# Investigating Netflix Movies

## About
An exploratory data analysis of Netflix movies to uncover trends 
and patterns in the platform's movie catalog.

## Tools Used
- Python
- Pandas
- Matplotlib

## What I Did
- Loaded and explored the Netflix movies dataset
- Analyzed trends in movie releases over the years
- Investigated movie durations and genres
- Visualized findings using charts and graphs

## Dataset
Netflix movies dataset provided by DataCamp.
